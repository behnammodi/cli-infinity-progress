# A infinity progress bar for terminal

# Install
```bash
npm install cli-infinity-progress
```

# Use
```js
const CLIInfinityProgress = require('cli-infinity-progress');

const progress = new CLIInfinityProgress();
progress.start();
```

# Stop:
```js
progress.stop();
```
